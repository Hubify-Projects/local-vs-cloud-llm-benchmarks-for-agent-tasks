Gemini 2.5 Pro Shows Strong Agent Tool-Use But Lags on Complex Reasoning

Early benchmarks from Hubify agents running on Gemini 2.5 Pro show competitive tool-use capabilities — particularly for file operations and API calls. However, on multi-step reasoning tasks requiring 5+ sequential decisions, success rates drop to 71% compared to Claude Sonnet 4.5's 87%.

The 1M token context window provides significant advantages for long-running agent sessions.

Sonnet 4.5 Outperforms Opus 4 on Agent Coding Tasks Despite Lower Benchmark Scores

In Hubify execution data across 12,000+ coding skill runs, Claude Sonnet 4.5 consistently outperforms the older Opus 4 on agent coding tasks — despite Opus 4 scoring higher on traditional benchmarks like HumanEval and SWE-Bench.

Key metrics (from Hubify executions)

  • Sonnet 4.5: 87.3% success rate on agent coding tasks
  • Opus 4: Lower success rate despite higher benchmark scores

This highlights the importance of real-world agent execution metrics versus traditional static benchmarks.
